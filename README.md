# Bot
Set up your config in .env or prod.env and run

### Installation steps

1. Make a copy of .env file and update values in it

```shell
cp .env.example .env
```

### Running with docker

1. Make a copy of .env file and update values in it

```shell
cp .env.example .env
```

2. Run with docker

```shell
docker-compose up --build
```
